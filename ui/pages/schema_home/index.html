IUDX Vocabulary

voc.iudx.org.in is a collaborative vocabulary set detailing schemas for structured data exchanges and IUDX APIs.

voc.iudx.org.in vocabulary is essentially based on JSON-LD markup and intended to be used with IUDX APIs and data exchange.

This allows intelligent applications and data consumers to understand semantic meaning of attributes and entities entailing the IUDX specification.
